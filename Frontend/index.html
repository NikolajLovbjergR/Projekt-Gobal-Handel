Hvordan handler Danmark med verden?

Få indsigt i Danmarks samhandel med omverdenen – se hvilke lande og varegrupper der betyder mest.

Danmarks største handelspartnere

Dette interaktive kort viser, hvilke lande Danmark handler mest med målt på import og eksport. Kortet er baseret på data fra 2018 til 2021 og fremhæver vores vigtigste handelspartnere såsom Tyskland, Sverige, USA og Kina. Klik på et land for at se handelsdata og sammenlign, hvordan Danmarks samhandel har udviklet sig over tid.

Samhandelspartner fordelt over år

Denne graf viser Danmarks import og eksport fordelt på de 9 lande, vi handler mest med. Hver søjle repræsenterer enten import eller eksport i det valgte år, så du nemt kan sammenligne, hvor meget vi køber og sælger. Grafen giver et tydeligt overblik over, hvor Danmark har handelsoverskud – og hvor vi importerer mere, end vi eksporterer. Hold musen over en søjle for at se den præcise værdi. Tree map over DKs Import og Eksport

Denne treemap-visualisering viser, hvilke typer varer Danmark handler mest med. Hver firkant repræsenterer en varekategori, og størrelsen viser, hvor stor en del af den samlede handel den udgør. Du kan hurtigt få overblik over, hvilke varer fylder mest i både vores import og eksport, og hvordan de enkelte sektorer bidrager til dansk samhandel.
